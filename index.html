Technical Features

High-performance tools integrated into one web-based platform.

Intrusion Detection

Native integration with Snort 3 and Zeek for real-time packet inspection and log generation.

ML Anomaly Detection

Using Python (Scikit-learn) to cluster log data and identify zero-day attack patterns.

React Visuals

Highly responsive React.js components including Recharts for live traffic visualization.

The Web Dashboard

Our web-based approach allows security analysts to monitor network health from anywhere and anytime. The frontend communicates via a Python FastAPI to fetch indexed logs from MongoDB.
