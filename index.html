Core Project Objectives

High-performance features aligned with modern cybersecurity requirements.

Traffic & Alert Analytics

Aggregates IDS data to provide a consolidated view of network frequencies and alert trends via dynamic timelines.

Intelligent behavior Filtering

Detects suspicious transactions using threshold rules for high-volume traffic and repeated authentication patterns.

ML Anomaly Detection

Leverages data mining and ML to recognize unusual patterns where traditional rules-based systems fall short.

Configurable Command Center

A fully interactive dashboard for administrators to monitor network health, configure threat parameters and select trained ML models.

  • ✅ Logical Quarantine: Track and manage malicious traffic from blacklisted domains.
  • ✅ Custom Definitions: Adjust thresholds for DoS and dangerous traffic definitions.
  • ✅ Forensic Search: Advanced search and filtering by category, time and protocol.
